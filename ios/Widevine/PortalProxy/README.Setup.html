Required RPMs
  • httpd
  • perl
  • perl-Compress-Zlib
  • perl-HTML-Tagset
  • perl-HTML-Parser
  • perl-libwww-perl
  • CypherDemo

Installation
  1. Copy /var/www/html/PortalProxy/widevine.conf into /etc/httpd/conf.d/.
  2. Restart httpd ( /etc/init.d/httpd restart )
  3. Copy /var/www/html/PortalProxy/GetEMMs.cgi into /usr/local/widevine/cypherpc/cgi-bin/. ( If the destination dir does not exist, use 'mkdir -p' to create it. )
